MetaPals Catch a Mole!

Guides

  • Time to finish: 1hr 30min.
  • Feel free to google.
  • Try this mini app and collect the requirements.
  • Feel free to ask the interviewer if you have any questions before and during working on this assignment.
  • Build the UI with vanilla javascript + CSS or SCSS, without React, Vue, Bootstrap, or any UI components library.
  • But using Tailwind CSS or similar CSS utilities is allowed.
  • Using assets bundler such as Vite or Webpack is allowed.
  • The time for the mole to hide itself is randomized between 200ms & 400ms.
  • There is "debounce" in the app, which is prevent the mole to come out from hole if you keep clicking on any hole rapidly.
  • The debounce function must be reusable
  • Submit by providing your GitHub repository information, include a README.md file and testable demo on GitHub pages.
